# Tetraonix Consulting Premium Website

Corporate consulting-first Next.js website for Tetraonix Consulting featuring:

- Tetraonix QA Platform
- Tetraonix HRMS
- AI automation consulting
- QA engineering modernization
- Cloud, cybersecurity, workforce, support, and managed services
- Attached Tetraonix icon and text-logo assets

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
npm start
```
